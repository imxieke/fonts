## Fonts

## mono
> 等宽字体（英語：Monospaced Font）是指字符宽度相同的电脑字体。
> 与此相对，字符宽度不尽相同的电脑字体称为比例字体

## Sans Serif(无衬线体) 和 Serif(有衬线体)
> 后者对于人眼的辨识来说会更轻松一些，所以阅读的时候会比较舒服。日本文字偏欧美的无衬线体（Sans-serif），所以大部分的人都使用歌德体(相当于西洋文字的无衬线体)。
